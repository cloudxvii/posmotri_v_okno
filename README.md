## «Посмотри в окно» — Видео-приложение

![Превью приложения](images/preview.png)
*Главный экран приложения с видео и карточками*

### Описание проекта

Интерактивное веб-приложение для просмотра видеозаписей с камер наблюдения из разных городов. Проект демонстрирует навыки CSS-стилизации сложного интерфейса с динамическим контентом.

### Цель проекта

Разработка стильного и функционального интерфейса для видео-приложения с возможностью поиска по различным параметрам. Основные задачи:
* Создание адаптивной вёрстки существующего HTML-макета
* Реализация кастомных элементов управления
* Стилизация интерактивных компонентов

### Технологии и инструменты

| Категория | Инструменты |
|-----------|-------------|
| HTML | Структурная разметка, шаблоны `<template>` |
| CSS | Flexbox, Grid Layout, анимации |
| Шрифты | Fira Sans Condensed, Oswald |
| Дополнительно | Кастомные формы, прелоадеры, обработка ошибок |

### Функциональность

1. **Основной видеоплеер** с автоматическим воспроизведением
2. **Система поиска** по городам и времени суток
3. **Список карточек** с превью видео
4. **Кастомные чекбоксы** для фильтрации
5. **Прелоадеры** для загрузки контента
6. **Обработка ошибок** при запросах

### Как запустить проект локально

```bash
# Клонируйте репозиторий
git clone [ссылка на репозиторий]

# Перейдите в папку проекта
cd посмотри-в-окно

# Откройте проект в браузере через index.html
```

### Особенности реализации

* **Интерактивные элементы**:
  * Кастомные чекбоксы
  * Стилизованные кнопки
  * Активное состояние карточек
  * Подсказки и ошибки

* **Компоненты интерфейса**:
  * Прелоадеры для загрузки
  * Модальные окна ошибок
  * Динамический список карточек
  * Форма поиска с валидацией

* **Адаптивность**:
  * Фиксированная вёрстка под десктоп
  * Оптимизированная компоновка элементов
  * Корректное отображение всех компонентов
